The JavaScript Dictionary

JavaScript (JS)

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

An assignment operator assigns a value to its left operand based on the value of its right operand.

Variables

Can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.

Variable naming conventions

  • Should begin with a letter
  • Case Sensitive
  • Should use clear names that describe the value being stored
  • Always name your variables in a manner that will be easier for other developers to understand

Functions

A set of statements that performs a task or calculates a value.

Methods

A function which is a property of an object.

Arguments

An Array like object accessible inside functions that contains the values of the arguments passed to that function.

Parameters

A named variable passed into a function.

Return

A statement that stops the execution of a function and returns a value from that function

Chaining methods

Also known as Cascading, means repeatedly calling one method after another on an object, in one continuous line of code.

Strings

An object that is used to represent and manipulate a sequence of characters.

Booleans

An object wrapper for a boolean value.

Undefined

A global property that represents the primitive value undefined. It is one of JavaScript's primitive types.

NaN

The global property value representing Not-A-Number.

Null

Represents the intentional absence of any object value. It is one of JavaScript's primitive values and is treated as falsy for boolean operations.

Alert

A method that displays an alert dialog with the optional specified content and an OK button.

Comments

An interface that represents textual notations within markup; although it is generally not visually shown, such comments are available to be read in the source view.

jQuery

A JavaScript Library that focuses on simplifying DOM manipulation, AJAX calls, and Event handling.

Attributes

A property that returns a live collection of all attribute nodes registered to the specified node.
